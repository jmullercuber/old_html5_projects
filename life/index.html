Working with the Canvas - Game of Life

Let's see what I can do with the new HTML5 canvas! As my first project, I have recreated Conway's Game of Life. I have already implemented a version in Khan Academy (Processing.js), and am converting this demonstration not only for portability, but also in joy of the Mozilla Webmaker Party!.

This project has given me an opportunity to utilize and improve my JavaScript, HTML and CSS skills. As a common lesson of programming, practically nothing works right the first time. My expanding knowledge of JavaScript and CSS is really what makes this possible, and is mostly what you see and interact with (not necessarily in that order). I probably should cite Stack Exchange and other web resources right now, and say that there are users' whose code I have modified to suit my purposes, but I will just provide the previous 36 words. Welcome everyone to my HTML5 implementation of Conway's Game of Life.

Controls:

Canvas must be selected (clicked or tabbed) before using the controls below!

  • Pause: P
  • Continue: C
  • Restart (Blank): R
  • Randomly Generate: G

  • Spacebar: Continuously turn the cells your mouse is hovering over to ON state in the next generation
  • Mouse Click: Dynamically turn the cell you're hovering over to ON state in the next generation
  • WASD: Horizontally and vertically translate the grid
